# TAICHI-flet
 基于flet的一款windows桌面应用，实现了爬取图片、音乐、小说、磁力链接的功能。

 特点：多功能娱乐软件，界面美观、简洁。
 
会持续更新

## 使用方法
方法1. 下载源代码，安装`requirements.txt`中的包，运行`run.py`文件。

方法2. 下载[release](https://github.com/cuifengcn/TAICHI-flet/releases/tag/publish)文件，解压后运行`太极.exe`文件
